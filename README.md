
**harnais\_map**: The Map Harness for the Harnais Family

The package tests Maps and their contents.

## Installation

Add **harnais\_map** to your map of dependencies in *mix.exs*:


    def deps do
      [{:harnais_map, "~> 0.1.0"}]
    end
